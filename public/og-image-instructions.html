Open Graph Image Instructions

How to Create an Open Graph Image for HukukArama

To improve social media sharing, please create an Open Graph image with the following specifications:

  1. Size: 1200 x 630 pixels (standard Open Graph size)
  2. Background: Dark background (black or dark gradient)
  3. Logo: Include the HukukArama logo prominently
  4. Text: "HukukArama - Yapay Zeka Destekli Hukuk Platformu"
  5. Colors: Use the brand gold color (#FFD613) for the main text/logo
  6. Save as: og-image.jpg in the public folder

Example SVG Code

Here's a simple SVG example that could be converted to a JPG:

<svg xmlns="http://www.w3.org/2000/svg" width="1200" height="630" viewBox="0 0 1200 630">
  <rect width="1200" height="630" fill="#000000"/>
  <text x="600" y="315" font-family="Arial" font-size="60" fill="#FFD613" text-anchor="middle">HukukArama</text>
  <text x="600" y="400" font-family="Arial" font-size="40" fill="#FFFFFF" text-anchor="middle">Yapay Zeka Destekli Hukuk Platformu</text>
</svg>

You can use a tool like Convertio to convert this SVG to JPG.

Note: Once created, make sure to update the Open Graph tags in index.html to point to the correct image path.
